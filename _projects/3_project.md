- **Overview:** This is a first-person perspective 3D Unity game. The game is designed as a horror, puzzle game which requires the player to explore the map and discover tools and puzzle in order to escape.

- **Game Mechanism:** We developed different stages and tools that players can interact with. By successful use of tools, the monster will be trapped and the player can escape safely.

- **Game Link:** [Doomed Deadlock](https://drive.google.com/file/d/1QGefzEiU41IujIHw28PHc-RG7J2g8zxd/view?usp=drive_link)
